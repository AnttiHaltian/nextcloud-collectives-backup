# nextcloud-collectives-backup
Experimental student project to design a backup process for the Nextcloud Collectives app.


**WARNING**

This repository contains an experimental student project.
It is NOT an officially supported Nextcloud solution and is NOT approved
for production use.

Do not use this code against production Nextcloud instances


## Purpose

The goal of this project is to design and implement a **backup process**
for content stored in the **Nextcloud Collectives** app.

The focus is on:
- Understanding where Collectives data lives logically
- Extracting relevant data via supported interfaces
- Designing a repeatable backup process


## Out of Scope

This project is NOT:
- A full Nextcloud backup solution
- A replacement for server or database backups
- A direct modification of Nextcloud core or the Collectives app
